# Página Interativa

Uma página web simples criada com HTML, CSS e JavaScript. Você pode personalizar o texto, as cores e a aparência com facilidade.

## Como usar

- Abra `index.html` no navegador.
- Edite o texto em `index.html` para mudar o conteúdo.
- Ajuste `styles.css` para trocar cores, fontes e layout.
- Modifique `script.js` para adicionar mais interações.

## Funcionalidades atuais

- Mudar cor de fundo com um botão.
- Alternar entre tema claro e escuro.
- Atualizar o título principal da página pelo campo de texto.

## Próximos passos

- Adicionar novas seções ou imagens.
- Criar um formulário de contato.
- Inserir links para redes sociais.
