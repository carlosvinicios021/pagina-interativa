# Página Interativa

Este é um projeto de portfólio simples em HTML, CSS e JavaScript, criado para demonstrar habilidades de frontend e apresentar um projeto funcional. O objetivo é mostrar como você pode construir uma página pessoal responsiva, interativa e fácil de personalizar.

## Sobre o projeto

- Projeto construído com HTML semântico, CSS moderno e JavaScript simples.
- Responsivo para telas menores com layout adaptável.
- Interatividade que permite modificar o título da página, trocar cor de fundo e alternar tema escuro.
- Ideal para iniciantes que desejam praticar e compartilhar um projeto real no GitHub.

## Funcionalidades

- Atualização dinâmica do título através de um campo de texto.
- Alternância de tema claro/escuro para melhorar a experiência do usuário.
- Controle de cor de fundo com feedback visual imediato.
- Layout em cartões (`cards`) com aparência limpa.
